ZeroToDeploy

ZeroToDeploy는 완전한 빈 상태에서 시작해, 아이디어를 웹 애플리케이션으로 구현하고 실서비스까지 배포하는 전 과정을 체계적으로 기록하고 공유하는 프로젝트 사용 기술 및 도구 백엔드: Spring Boot 데이터베이스: MySQL 서버 환경: WSL Ubuntu, Apache, Tomcat 프로젝트 창시자 소개 안녕하세요! 저는 웹 애플리케이션 개발과 배포에 열정을 가진 개발자입니다. ZeroToDeploy는 제 기술 여정을 공유하고, 실무에 필요한 지식을 나누며, 같은 목표를 가진 개발자들과 성장하기 위해 시작한 프로젝트입니다. 이곳에서 저는 도전과 배움을 통해 어떻게 서비스가 탄생하고 성장하는지를 보여드릴 예정입니다. 주요 목표 초보자도 따라할 수 있는 단계별 배포 과정 정리 문제 해결 및 노하우 기록 최종적으로, 직접 운영 가능한 서비스 완성
