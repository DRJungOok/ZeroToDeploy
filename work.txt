1. 로그인 기능
    - 사용자 아이디 입력
    - 비밀번호 입력
    - 입력된 정보 검증
    - 로그인 성공/실패 메시지 출력

2. 회원가입 기능
    - 사용자 아이디 입력
    - 비밀번호 입력
    - 이메일 입력
    - 입력된 정보 저장
    - 회원가입 성공/실패 메시지 출력

3. 소셜 로그인 기능
    - 소셜 로그인 버튼 (예: 구글, 페이스북)
    - 소셜 로그인 API 연동
    - 소셜 로그인 성공/실패 메시지 출력

4. 글쓰기 기능
    - 제목 입력
    - 내용 입력
    - 작성된 글 저장
    - 글쓰기 성공/실패 메시지 출력
